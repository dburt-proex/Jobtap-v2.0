### Jobtap-v2.0
Automated lead generation for contractors, and an app for homeowners to find trusted, verified help within their homes and community.

## Jobtap Admin Portal

A production-ready admin dashboard for managing the Jobtap platform — contractors, homeowners, jobs, and leads.

### Getting Started

```bash
cd jobtap-admin
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and sign in with the demo credentials.

**Demo Login:** `admin@jobtap.com` / `admin123`

### Features

- **Authentication** — Secure login/logout with cookie-based session management and protected routes
- **Dashboard** — Key metrics overview with stats cards, recent jobs, and recent leads tables
- **Contractors** — View, search, and filter all registered contractors with ratings and verification status
- **Homeowners** — Manage homeowner accounts and track their activity
- **Jobs** — Track all jobs with status filtering, category search, and budget tracking
- **Leads** — Monitor lead pipeline with conversion tracking and source attribution

### Tech Stack

- **Next.js 16** (App Router, Turbopack)
- **TypeScript**
- **Tailwind CSS 4**
- **React 19**
